A - 回転操作/Rotation

Time limit時間制限 : 2sec / Stack limitスタック制限 : 256MB / Memory limitメモリ制限 : 256MB

問題文

長さ N の文字列 S=s_{1}s_{2}...s_{N} があります．この文字列に，以下の操作(この問題文では「回転操作」と呼ぶ)を決められた順番で M 回行うことを考えます．

  • 3 つの整数 L,R,K (1≦L≦R≦N,1≦K≦100) が与えられるので，文字列 S の閉区間 [L,R] の部分文字列を K 文字右方向に回転させたものを新たな S とする．具体的には，s_{L}s_{L+1}...s_{R-1}s_{R} と並んでいる部分文字列を s_{R}s_{L}s_{L+1}...s_{R-1} という並びにするという操作を K 回繰り返す．当然，区間に含まれない文字列の並びは変化しない．

あなたには，その M 個の回転操作が与えられるので，それらを与えられる順番に文字列 S に適用した結果，最終的に得られる文字列を出力してください．

入力

入力は以下の形式で標準入力から与えられる．

N
S
M
L_1 R_1 K_1
L_2 R_2 K_2
:
L_M R_M K_M
  • 1 行目には，文字列の長さを表す整数 N (1 ≦ N ≦ 100) が与えられる．
  • 2 行目には，文字列 S (S の長さは N に等しい) が与えられる． S は 半角アルファべット小文字(a-z)のみから成る．
  • 3 行目には，回転操作の回数を表す整数 M (1 ≦ M ≦ 100) が与えられる．
  • 4 行目から M 行には，一連の回転操作の情報が与えられる． そのうち i 行目には，i 番目に行う回転操作における L,R,K を表す 3 つの整数 L_i,R_i,K_i (1≦L_i≦R_i≦N,1≦K_i≦100) が与えられる．

出力

文字列Sに与えられた回転操作を順番に適用し、その結果を出力してください。

末尾の改行を忘れないこと．

入力例1

abcde
2
2 4 1
1 5 2

出力例1

ceadb

1 回目の操作の後，文字列は adbce となります．さらに， 2 回目の操作の後，文字列は ceadb となります．

Question

There is a string S=s_{1}s_{2}...s_{N} with length N. To this string, suppose we are going to perform the following operation (“Rotation” in the rest of this question), M times with a given order.

  • The following 3 integers will be given: L, R, K (1≦L≦R≦N, 1≦K≦100). We will rotate the substring s_{L}s_{L+1}...s_{R-1}s_{R} in this string to the right for K letters. Specifically, we will rearrange the part of the string which is aligned as s_{L}s_{L+1}...s_{R-1}s_{R} to s_{R}s_{L}s_{L+1}...s_{R-1} K times. This will not change the arrangement of the characters that are not included to the interval.

You will be given M Rotations. Output the string that you will get after applying them to the string S．

Input

Input will be given in the following format from Standard Input: 

N
S
M
L_1 R_1 K_1
L_2 R_2 K_2
:
L_M R_M K_M
  • To the 1st line, integer N (1 ≦ N ≦ 100), which represents the length of the string, will be given.
  • To the 2nd line, the string S (length of S equals N) will be given. S consist only lowercase letters (a-z).
  • To the 3rd line, integer M (1≦M≦100), which represents the number of times of rotation, will be given.
  • On M lines after 4th line, the information of these series of rotation will be given. To ith line among them, 3 integers L_i, R_i, K_i (1≦L_i≦R_i≦N, 1≦K_i≦100), which represents L, R, K on rotation that will take place on for the ith time.

Output

Output the result obtained after applying the M rotations to the string S.

Make sure to insert a line break at the end of the output.

Input Example #1

5
abcde
2
2 4 1
1 5 2

Output Example #1

ceadb

After the first operation, the string will be adbce. Then, after the second operation, the string will be ceadb.
